The Component Library imported in KMgen

Main window

  • The left pane of the KMgen main window displays the selected hierarchy of frames (there are 6 hierarchies: Classes and intances, Slot classes and instances, ... and Situation instances).
  • The definition of a frame is displayed in a HTML browser (top-right pane of the KMgen main window). Clicking on a link (any coloured word) brings a menu like the one showed below.
  • Editing occurs in Emacs, using a specific KM mode (indentation, coloration and word completion), with the exception of manipulations of hierarchies which are done using commands like Add parent, Move, ...
